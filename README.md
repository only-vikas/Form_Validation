## Installation

### Prerequisites

- Node.js (v14.0.0 or higher)
- npm or yarn

### Setup

```bash
# Clone the repository
git clone https://github.com/only-vikas/Form_Validation.git

# Navigate to the project directory
cd Form_Validation

# Install dependencies
npm install
# or
yarn install
